# BookStore
BOOKSTORE MANAGEMENT SYSTEM PROJECT TYBCA(THIRD YEAR OF BCA SIXTH SEMESTER PROJECT)

PROJECT TECHNOLOGIES (LANGUAGES) USE
1. REACTJS
2. REACT ROUTER DOM
3. REACT SLICK
4. MATERIAL UI
5. MUI-ICONS

BACKEND
1. BCRYPTJS
2. EXPRESSJS
3. EXPRESS VALIDATOR
4. JSON WEB TOKEN
5. MONGOOSE
6. MULTER
7. NODEMON
